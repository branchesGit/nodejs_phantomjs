# 生成图标类
> 本工程旨在调用java端API接口生成相应的图表信息
> 主要图表信息分为：
* 柱状图 (现阶段需求)
* 表格 （现阶段需求）
* 折线图
* 饼图

# 运行
1. `$ npm install`

2. 配置启动文件，如图所示：
